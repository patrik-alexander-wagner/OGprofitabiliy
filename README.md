# OGame ROI Advisor

A Tampermonkey script for OGame that calculates the Return on Investment (ROI) for Mines, Lifeform Buildings, and Technologies to help you decide the most profitable next step.

## Features
- **Top 20 Investments**: Shows a ranked list of upgrades based on payback time.
- **Lifeform Support**: Includes calculations for Rock'tal buildings (Magma Forge, Crystal Refinery, Deuterium Synthesizer).
- **Fallback Mode**: Scans the current page if the API is blocked.
- **Safe**: Does not automate actions, only strictly calculates and displays data.

## Installation
1. Install [Tampermonkey](https://www.tampermonkey.net/).
2. Drag and drop `roi_advisor.user.js` into the extension dashboard or create a new script and copy the contents.

## Usage
Click the "ROI Advisor" button in the OGame sidebar menu.
